# GEGP

__Implementation of Gradients-Enhanced Kriging in `stan`.__

At the moment:
* Noisy data and gradients    
* Gaussian covariance. See _e.g._ [Ulaganathan (2016) Eng. Comp. 32:15-34](https://doi.org/10.1007%2Fs00366-015-0397-y)
